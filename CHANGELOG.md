# Changelog

All notable changes to video-to-notes will be documented in this file.

## 0.1.0 - 2026-07-08

Initial public release.

### Added

- Caption-first video transcript workflow using `yt-dlp`.
- Local Whisper fallback for videos without usable captions.
- Searchable local HTML reports and dashboard indexing.
- Optional Notion publishing.
- Optional Obsidian publishing.
- Codex skill wrapper for repeatable video research workflows.
