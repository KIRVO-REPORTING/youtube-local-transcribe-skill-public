video-to-notes

video-to-notes is a local-first Python CLI that turns YouTube, Bilibili, TED, and other yt-dlp video URLs into transcripts, timestamped AI summaries, searchable HTML reports, and optional Notion or Obsidian notes.

Use cases

YouTube transcript to notes
Bilibili video transcript and summary
Local Whisper fallback transcription
Timestamped AI summaries
Notion and Obsidian video knowledge base
Codex skill for video research workflows

Quick start

git clone https://github.com/KIRVO-REPORTING/video-to-notes.git
cd video-to-notes
./install.sh
video-to-notes configure
video-to-notes process "VIDEO_URL"

Why caption-first?

Many videos already include usable captions. video-to-notes uses captions first and only runs local Whisper when captions are missing or unsuitable.
